# SQL-Sales-Analysis-Project
## Project Overview
This project analyzes retail sales data using SQL to uncover business insights related to revenue, product performance, and customer trends.

## Objectives
- Analyze sales and profit trends
- Identify top-performing products and categories
- Understand regional performance
- Use advanced SQL techniques like window functions

## Tools Used
- SQL Server (SSMS)
- SQL (Joins, Aggregations, Window Functions)

## Key Insights
- Technology category generates the highest revenue
- West region contributes the most to overall sales
- Top products drive a significant portion of total revenue
- High discounts negatively impact profitability

## SQL Techniques Used
- GROUP BY & Aggregations
- RANK() and PARTITION BY
- Window Functions (LAG, Running Total)

## Conclusion
SQL analysis helps uncover critical business patterns and supports data-driven decision-making.
